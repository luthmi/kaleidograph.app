# Harmonic Kaleidograph Website

Landing page for The Harmonic Kaleidograph — a synesthetic visualization engine that transforms MIDI into living geometry.

## Structure

```
├── index.html          # Main landing page
├── CNAME               # Custom domain config for GitHub Pages
├── images/

│   └── hk-icon.jpg                 # Nav icon
├── screenshots/        # Gallery screenshots (add your own)
└── demo/               # Web demo (add later)
```

## Setup

This site is hosted via GitHub Pages. To update, push changes to the `main` branch.
